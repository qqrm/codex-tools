# Developer Experience Review Skill

## What It Is
A reusable skill for reducing friction in setup, feedback loops, command surfaces, and day-to-day contributor workflows.

## When To Use
- Local setup is confusing or slow.
- Contributors repeat manual steps.
- Build, test, or lint loops feel heavier than they should.

## Default Persona
Use the `delivery_engineer` persona first. Bring in `analyst` when the review must prioritize user pain and adoption impact.
